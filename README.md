HRMS - Human Resource Management System
A full-stack HRMS application for managing employees, teams, and organisations with comprehensive audit logging.

Features
Organisation Management: Multi-tenant architecture

Employee Management: Full CRUD operations for employees

Team Management: Create and manage teams

Team Assignments: Many-to-many employee-team relationships

Authentication: JWT-based secure authentication

Audit Logging: Comprehensive activity tracking with filtering

Responsive UI: Modern React frontend

Tech Stack
Frontend: React, Vite, CSS3

Backend: Node.js, Express.js, Sequelize ORM

Database: MySQL

Authentication: JWT, bcrypt

Containerization: Docker & Docker Compose
